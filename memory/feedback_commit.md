---
name: No hacer commits automáticamente
description: El usuario prefiere hacer los commits él mismo, solo pedir el mensaje
type: feedback
---

No ejecutar git add ni git commit. Solo redactar el mensaje de commit cuando el usuario lo pida.

**Why:** El usuario quiere controlar qué se commitea y cuándo.
**How to apply:** Cuando pidan "armame el commit message", solo escribir el texto del mensaje, no ejecutar ningún comando git.
